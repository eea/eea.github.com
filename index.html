Featured EEA products (Zope/Plone add-ons)¶

  • EEA Faceted Navigation

A very powerful interface to improve search within large collections of items.

  • EEA Daviz

The first semantic web data visualization tool for Plone

  • EEA Annotator

A powerful tool that replicates collaborative inline comments from word in Plone.

  • EEA Progress Bar

Progress bar based on document’s workflow states

  • EEA Tags

A Google+ / Facebook like replacement for the Plone keywords widget.

  • EEA Uberlisting

Enhances and extends the listing layouts capabilities of Plone

  • EEA Depiction

A generic system for creating thumbnails representations for content-types

  • EEA Alchemy

Auto-discover and auto-highlight keywords, geographical locations and temporal coverage.

  • EEA Relations

A tool to define possible relations between objects.

  • EEA ElasticSearch

An elasticsearch setup suite for indexing resources in the EEA Semantic Data Service.
